Awesome MCP runs a Model Context Protocol (MCP) server directly inside Burp Suite and exposes agent-friendly tools for real pentest workflows.

It is designed for human + AI agent collaboration with stable IDs, strict JSON contracts, safe body serialization, and practical integrations with Burp history, Repeater, Intruder, scope and Collaborator.

Key Features

  • Stable Proxy history IDs and Site Map keys for deterministic follow-up actions
  • Pagination-safe query responses: total, next, results
  • Smart body serialization: omit large text by default, optional truncation, binary omitted by default
  • Bulk HTTP/1.1 and HTTP/2 request execution with optional parallel mode and rate limiting
  • Send requests to Repeater and Intruder, including Intruder insertion-point templates
  • Scope management tools, including optional subdomain scope rule updates
  • Cookie Jar query/set/expire tools for session-aware agent workflows
  • Collaborator tooling and scanner issue querying (Burp Professional only)

Usage

  1. Open the Awesome MCP Burp tab and enable the server with the toggle.
  2. Set host, port and transport; click Apply to apply endpoint settings.
  3. Copy endpoint URL and connect your MCP client.

Default endpoint (streamable_http): http://127.0.0.1:26001/mcp
Optional SSE endpoint: http://127.0.0.1:26001

For protocol details see modelcontextprotocol.io.
